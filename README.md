# Unity Tech Hub x Agilyti - Data Engineering Training

## Overview

This repository contains my learning progress, exercises, and projects completed during the Unity Tech Hub x Agilyti Data Engineering Training program.

The training is organized into weekly modules, with 5 practice days each week. Each day focuses on different data engineering concepts, gradually building skills required to work with real-world data pipelines.

## Training Structure

The program follows a weekly learning structure:

- Each week contains 5 training days.
- Each day includes practical exercises, coding tasks, and documentation.
- Tasks build from basic concepts into more advanced data processing and engineering workflows.
