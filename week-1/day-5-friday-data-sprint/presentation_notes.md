## 1. What dataset did we analyze?

We analyzed an order dataset from an electronics and office-supplies business.
The dataset contains information about customers, cities, products, categories, quantities, prices, statuses, and order dates.

## 2. What did we do in Python?

In Python, we:
- Read the orders.csv file.
- Created functions to analyze the data.
- Counted total orders.
- Filtered completed, pending, and cancelled orders.
- Calculated total amounts.
- Calculated completed revenue.
- Counted orders by status, city, and category.
- Generated a business report.

## 3. What did we do in SQL?

In SQL, we:
- Created the orders table.
- Inserted the same dataset from CSV.
- Selected and filtered orders.
- Calculated total_amount.
- Calculated completed revenue.
- Used COUNT and GROUP BY to analyze statuses, cities, and categories.
- Found the most valuable orders.

## 4. What was our most important business insight?

Our most important insight was identifying which city and category performed best and understanding which orders generated real revenue.

## 5. What was difficult?

The difficult parts were keeping the Python and SQL results consistent, organizing the code with functions, and making sure calculations were correct.

## 6. What would we improve if we had more time?

If we had more time, we would add more data visualization, deeper analysis, and create a dashboard to present the results better.
