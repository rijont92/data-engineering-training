# Monday Practice - Python Refresh & Data Processing

This folder contains Python exercises focused on solidifying Python basics for Data Engineering.

## Topics Covered
- Lists of Dictionaries representation of datasets
- Iteration using basic loops
- Conditional filtering and custom categorization rules
- Statistical aggregations (totals, averages, groupings)
- Program structure using modular functions
- Safe handlers for empty inputs

## Project Structure
- `monday_practice.py`: Contains basic processing tasks (Detyra 1 to 5) executed sequentially.
- `student_report.py`: Contains modular report generator with helper functions and bonus task solutions (sorting, dynamic addition, top list, and empty check safety).
- `daily_report.txt`: A concise text summary of today's learning and completed tasks.
