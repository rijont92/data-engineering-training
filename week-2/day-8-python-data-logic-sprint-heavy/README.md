# Day 8 - Python Data Logic Sprint Heavy Version

## Business Scenario

This project simulates a business order data processing system. The goal is to analyze raw order data, validate the information, clean inconsistent values, and generate useful business reports. The pipeline helps transform raw data into reliable insights for decision-making.

## Files and Folders Included

- `python_data_analysis.py` - Main Python script that processes and analyzes the data.
- `order_data.py` - Contains the raw order dataset.
- `output/` - Folder where generated reports are saved.
  - `invalid_records.txt`
  - `validation_report.txt`
  - `business_report.txt`
- `README.md` - Project documentation.
- `logic_explanations.md` - Explanation of the program logic.
- `daily_report.txt` - Daily progress report.

## How to Run

Run the main Python file using:

```bash
python python_data_analysis.py
